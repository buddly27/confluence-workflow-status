# Workflow Status

This [Confluence macro](https://support.atlassian.com/confluence-cloud/docs/what-are-macros/) defines intuitive and descriptive statuses to accurately reflect the current state of a task.

![image](./demo.gif)

It differs from the standard [Status Macro](https://confluence.atlassian.com/doc/status-macro-223222355.html), which requires the user to type the label and define the color for each chosen status.
